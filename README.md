# Gym-Management-System
DBMS project for gym management including users, subscriptions, workouts, and payments.


## Features
- User management (admin, trainer, member)
- Subscription & plans
- Workout assignment
- Attendance tracking
- Payment system

## Tech Stack
- Frontend: React / JS
- Backend: Express (if used)
- Database: PostgreSQL

## Database

- schema.sql → Logical database design used for ER diagram
- schema_dump.sql → PostgreSQL generated schema (actual DB structure)

## Setup Instructions
1. Clone repo
2. Install dependencies
3. Setup database
4. Run project
